# 🍸 Austin Bar CRM

Mobile-friendly CRM for tracking bar visits, contacts, and vending machine placement outcomes.

**Live App:** [View on GitHub Pages](https://YOUR_USERNAME.github.io/bar-crm/bar-crm-app.html)

## Features

- 📍 Interactive map with 92+ Austin bars
- ✅ Quick outcome tracking (interested/not interested)
- 📝 Full CRM fields (contact name, role, notes, next actions)
- 📱 Mobile-optimized design
- 🎤 Voice-to-CRM workflow (update via voice messages)

## Tech Stack

- Vanilla JavaScript (no frameworks)
- Leaflet.js + OpenStreetMap (free maps)
- SerpAPI for bar data fetching
- GitHub Pages hosting

## Data Source

Bar data fetched from Google Maps via SerpAPI. Updated regularly.

---

Built for canvassing bars in Austin, TX for fragrance vending machine placements.
